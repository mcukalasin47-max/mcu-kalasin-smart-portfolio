# MCU Kalasin Smart Portfolio

ระบบแฟ้มสะสมผลงานและประเมินผลการปฏิบัติงานบุคลากร หน่วยวิทยบริการ วิทยาลัยสงฆ์ขอนแก่น จังหวัดกาฬสินธุ์

## ความสามารถชุดแรก
- Dashboard และความครบถ้วนตามพันธกิจ
- แฟ้มผลงาน รอบประเมิน รายงาน SAR และหน้ากรรมการ
- ทำเนียบบุคลากรและหน้า Portfolio
- ค้นหา เพิ่มผลงาน และรองรับมือถือ
- เตรียมพร้อมสำหรับ Vercel

## เริ่มต้น
```bash
npm install
npm run dev
```

## ข้อมูลจริงในระยะต่อไป
เชื่อม Google Workspace, Google Sheets และ Google Drive ผ่าน Google Apps Script โดยเก็บ URL และสิทธิ์ไว้ใน Environment Variables ของ Vercel ห้ามบันทึกข้อมูลลับลง GitHub
