# MCU Kalasin Smart Portfolio v2.0

ระบบแฟ้มสะสมผลงานและประเมินผลการปฏิบัติงานบุคลากร หน่วยวิทยบริการ วิทยาลัยสงฆ์ขอนแก่น จังหวัดกาฬสินธุ์

## ความสามารถ

- Dashboard และแฟ้มผลงานออนไลน์
- ทำเนียบบุคลากร 25 คนจาก Google Sheets
- เข้าสู่ระบบด้วย Google Identity Services
- บุคลากรแก้ไขได้เฉพาะโปรไฟล์ของตนเอง
- ผู้ดูแลระบบสามารถจัดการข้อมูลทุกคน
- บันทึกประวัติการเข้าสู่ระบบและการแก้ไขลง `AuditLog`
- SweetAlert2 สำหรับการยืนยัน สำเร็จ และข้อผิดพลาด
- Responsive สำหรับคอมพิวเตอร์ แท็บเล็ต และมือถือ

## ฐานข้อมูล

Google Sheets: `MCU Kalasin Smart Portfolio Database`

- Personnel
- Works
- Evidence
- EvaluationRounds
- Indicators
- Reviews
- Users
- Config
- AuditLog

## ติดตั้ง

```bash
npm install
npm run dev
npm run build
```

ดูการตั้งค่าล็อกอินและ API ที่ `VERCEL-SETUP.md`

## สิทธิ์แก้ไขข้อมูล

- `PERSONNEL`: แก้คำนำหน้า ชื่อ หมายเลขโทรศัพท์ และ URL รูปของตนเอง
- `ADMIN`: แก้ไขข้อมูลบุคลากรทุกคน
- อีเมลเข้าสู่ระบบ ตำแหน่ง ฝ่ายงาน บทบาท และสถานะบัญชีแก้โดยผู้ดูแลเท่านั้น

พัฒนาโดย พระมหาธงชัย วิลาสินี นักวิชาการศึกษา หน่วยวิทยบริการ วิทยาลัยสงฆ์ขอนแก่น จังหวัดกาฬสินธุ์
